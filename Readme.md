


# The Matrix

This is a fun program which creates a matrix like screen on your terminal.

## Getting Started

Just install curses library and make a local copy of this repository. 


### Prerequisites

Requires c++ along with ncurses library

### Running

compile with
g++ matrix.cpp -lncurses

run with
./a.out
